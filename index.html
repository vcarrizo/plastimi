Períodos disponibles

Agregar un nuevo período

Ejecutá procesar_mes.py con los archivos XLS del mes:

python procesar_mes.py \
  --rpcp199 rpcp199_XXXXXX.xlsx \
  --ragr004 ragr004_XXXXXX.xlsx \
  --catalogo "Productos Iniflex.xlsx" \
  --periodo 2026-06

Luego subí los datos al repo:

git add datos/
git commit -m "datos: agregar período 2026-06"
git push

Los datos estarán disponibles en pocos minutos cuando GitHub Pages actualice el sitio.
